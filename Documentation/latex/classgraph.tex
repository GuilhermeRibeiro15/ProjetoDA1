{\footnotesize\ttfamily double graph\+::edmonds\+Karp (\begin{DoxyParamCaption}\item[{int}]{source,  }\item[{int}]{target }\end{DoxyParamCaption})}



Applies the Edmonds-\/\+Karp algorithm to find the maximum flow in the graph. 


\begin{DoxyParams}{Parameters}
{\em source} & Index of the source node \\
\hline
{\em target} & Index of the target node \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
double Maximum flow value in the graph
\end{DoxyReturn}
The function initializes the flow of each edge to zero and repeatedly finds an augmenting path from the source to the target using the {\ttfamily find\+Path} function. For each augmenting path, it computes the minimum residual capacity along the path and updates the flow along the path accordingly. The process continues until there are no more augmenting paths. Time complexity\+: O(\+V $\ast$ E$^\wedge$2), where V is the number of stations and E is the number of tracks in the graph, due to the use of the Edmonds-\/\+Karp algorithm to find the maximum flow in the graph.

\begin{DoxyNote}{Note}
This function assumes that the graph is a directed graph with edges representing the capacity of the flow from the origin to the destination. 
\end{DoxyNote}

{\footnotesize\ttfamily void graph\+::find\+Max\+Flow\+District (\begin{DoxyParamCaption}\item[{int}]{k }\end{DoxyParamCaption})}



Finds the maximum flow for the top k districts and municipalities. 

This function first creates maps of all districts and municipalities in the graph with their corresponding stations. It then computes the maximum flow between all pairs of stations for each district and municipality, and stores the results in sorted order in vectors. The top k results from each vector are printed to standard output. Time complexity\+: O(\+V$^\wedge$2 $\ast$ E$^\wedge$4), where V is the number of stations in the graph and E is the number of tracks in the graph

{\footnotesize\ttfamily int graph\+::target\+Max\+Flow (\begin{DoxyParamCaption}\item[{int}]{target }\end{DoxyParamCaption})}



Computes the maximum flow from the source to a specific target in the graph. 

To find the maximum flow, an auxiliary node is added to the graph and all nodes with only one neighbor are connected to the auxiliary node with a standard track of infinite capacity. The maximum flow is then calculated from the auxiliary node to the target node using the Edmonds-\/\+Karp algorithm. Time complexity\+: O(\+E $\ast$ f) where E is the number of tracks in the graph and f is the maximum flow

{\footnotesize\ttfamily void Altered\+Trip\+Manager\+::find\+Maximum\+Flow\+Altered\+Network (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



This method prompts the user to input the origin and destination stations, and then calculates the maximum number of trains that can travel between them in the altered network using the Edmonds-\/\+Karp algorithm. 

The method uses the \mbox{\hyperlink{class_altered_trip_manager_a1bc4e699869b1baab1161ff2417a44e0}{find\+Station\+In\+Altered\+Hashtable()}} method to verify if the input stations are valid. If the user enters \char`\"{}\+Quit\char`\"{} at any point, the method returns. If the stations are valid, the method calculates the maximum flow between them using the edmonds\+Karp() method of the Altered\+Graph class. The total flow value is then printed to the console. \begin{DoxyNote}{Note}
The Edmonds-\/\+Karp algorithm is used to find the maximum flow in a network. It runs in O(\+V$\ast$\+E$^\wedge$2) time, where V is the number of vertices and E is the number of edges in the network. 
\end{DoxyNote}

This function reads the stations and tracks data from two CSV files and creates the corresponding objects. 

It opens and reads the \char`\"{}stations.\+csv\char`\"{} file and creates a \mbox{\hyperlink{class_station}{Station}} object for each line, then adds it to the graph data structure. Then, it opens and reads the \char`\"{}network.\+csv\char`\"{} file and creates a \mbox{\hyperlink{class_track}{Track}} object for each line, linking the corresponding \mbox{\hyperlink{class_station}{Station}} objects. The function also adds the newly created \mbox{\hyperlink{class_station}{Station}} objects to a hash table for easier lookup. \begin{DoxyNote}{Note}
If either file cannot be opened, the function prints a \char`\"{}\+File not found\char`\"{} message and returns. 
\end{DoxyNote}
